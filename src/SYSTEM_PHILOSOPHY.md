# SRIKA Living System Philosophy

## What Makes This Different

### Not a Website — A Living System

This is not a traditional website with sections, cards, and scroll-triggered animations. It's a continuous intelligent surface that responds to presence.

### Core Principles

1. **Everything Always Exists**
   - No `display: none`
   - No visibility triggers
   - Elements exist in different states of focus, not existence

2. **Flow Field Motion**
   - All elements respond to the same invisible force
   - Distance from "focus band" determines state
   - Opacity range: 0.3 to 1 (never fully invisible)
   - Blur range: 0 to 12px
   - Scale range: 0.96 to 1

3. **No Sections**
   - Hero drifts sideways, becomes navigation anchor
   - Features float in continuous space
   - Body→Input morphs across multiple scroll lengths
   - Time exists as density, not timeline
   - Pricing is access levels, not tables
   - Site dissolves rather than ends

4. **Material Language**
   - No glass effects
   - No cards or boxes
   - Fields, planes, layers
   - Depth via motion, not shadows

### Motion Philosophy

- **Field distortion** instead of slide-ins
- **Focus gravity** instead of pinning
- **Motion inheritance** instead of independent animations
- **Continuity** instead of discrete states

### Technical Implementation

- FlowFieldContext: Global scroll state
- calculateFocusDistance: Element positioning relative to focus
- getElementState: Continuous state interpolation
- All components respond to same flow field
- Fully reversible at any scroll speed
